# IoT26-HW04: Raspberry Pi Web Server using Flask

## Objective
Use RPI to create a standalone web server with a Raspberry Pi that can toggle two LEDs. 

## Source Code
* **`app.py`**: Flask 백엔드 서버를 구동하고, `lgpio` 라이브러리를 사용해 라즈베리 파이의 GPIO 17번, 27번 핀을 직접 제어하는 파이썬 스크립트입니다.
* **`templates/main.html`**: 사용자가 브라우저에서 접속했을 때 보이는 화면으로, 버튼 클릭 시 핀 상태를 변경하는 프론트엔드 UI입니다.
